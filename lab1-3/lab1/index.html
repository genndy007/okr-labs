Види автомобілів

Автомобі́ль — самохідна колісна машина, яка приводиться в рух встановленим у неї двигуном і призначена для перевезення людей, вантажу, буксирування транспортних засобів, виконання спеціальних робіт та перевезення спеціального устаткування безрейковими дорогами. Пересувається переважно суходолом.

Запчастини:

Двигун
Енергосилова машина, що перетворює який-небудь вид енергії на механічну роботу.
Колі́нчастий вал
Bал (чи вузол у випадку складеного вала) складної форми, призначений для перетворення зворотно-поступального руху (наприклад, поршня) в обертальний навколо своєї осі

Сучасні автомобілі поділяються на:

  • Tранспортні — основна група, призначені для перевезення людей і вантажу
  • Bантажні
  • Пасажирські
  • Легкові
  • Мікроавтобуси
  • Автобуси
  • Гоночні
